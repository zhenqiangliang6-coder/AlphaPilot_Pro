# 自动化调度器 - 快速参考卡

## 🚀 一键启动

```
双击运行：D:\AlphaPilot_Pro\启动自动调度.bat
```

---

## ⏰ 执行时间

- **每天**：工作日（周一到周五）
- **时间**：早上 8:30
- **跳过**：周六、周日

---

## 📋 执行顺序与间隔

```
8:30 → 1_启动QMT.bat      (启动 QMT 交易终端)
        ↓ 
        ⏱️ 等待 90 秒（确保 QMT 完全就绪）
        ↓
     → 2_启动监听.bat      (启动信号监听)
        ↓
        ⏱️ 等待 90 秒（确保监听模块完全就绪）
        ↓
     → 3_AlphaPilot.bat    (启动主策略)
        ↓
     ✅ 全部完成
```

**总耗时**：约 3 分钟（90 秒 × 2 次间隔）

---

## 🔍 为什么需要 90 秒间隔？

### 1️⃣ **QMT 启动阶段（90 秒）**
- 加载交易接口模块
- 建立与券商服务器的连接
- 初始化行情数据订阅
- 验证账户信息和权限

### 2️⃣ **信号监听阶段（90 秒）**
- 建立邮件服务器连接
- 开始监听信号文件目录
- 初始化信号解析器
- 测试信号处理流程

### 3️⃣ **主策略阶段（无需等待）**
- 自动检测 QMT 和监听模块状态
- 初始化所有策略模块
- 开始执行交易逻辑

## ⚠️ 重要提醒

1. ✅ **保持窗口打开**：调度器必须持续运行
2. ✅ **最小化窗口**：不要关闭，可以最小化
3. ❌ **禁止 Ctrl+C**：这会停止调度器
4. 🔄 **开机自启**：可添加到启动文件夹

---

## 🔧 修改执行时间

编辑 `auto_scheduler.py` 第 18-19 行：

```python
SCHEDULE_HOUR = 8      # 改为其他小时（0-23）
SCHEDULE_MINUTE = 30   # 改为其他分钟（0-59）
```
