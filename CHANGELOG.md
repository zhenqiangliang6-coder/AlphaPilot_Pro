# AlphaPilot Pro 版本更新日志

## V8.95 (2026-04-15) - 动态止损保守版升级

### 🎉 重大更新

#### ✅ 核心功能：动态分级止损系统（保守版）

**调整内容**：
- 从平衡版参数调整为更敏感的保守版参数
- 适用于风险厌恶型投资者和震荡/弱势市场

**新参数**：

1. **监控阶段（-0.8%）** ⬅️ 从-1.5%提前
   - 当亏损达到-0.8%时，股票进入监控观察名单
   - 记录最低盈亏比例和监控开始时间
   - 不执行任何卖出操作，仅预警

2. **一级止损（-1.7%）** ⬅️ 从-2.5%提前
   - 触发条件：亏损达到-1.7%
   - 执行动作：卖出50%仓位
   - 目的：快速控制风险敞口

3. **二级止损（-3.5%）** ⬅️ 从-4.0%提前
   - 触发条件：亏损达到-3.5%
   - 执行动作：全部清仓
   - 目的：严格止损，将最大亏损控制在3.5%以内

4. **反弹保护机制** ✅ 保持不变
   - 触发条件：已触发一级止损后，股价反弹超过成本价（盈利>0%）
   - 执行动作：重置所有止损状态
   - 价值：避免"卖在最低点"，给予优质股票二次机会

## V8.9 (2026-04-15) - 动态分级止损升级

### 🎉 重大更新

#### ✅ 核心功能：动态分级止损系统

**替换内容**：
- ❌ 移除：单一阈值硬性止损（`STOP_LOSS_RATIO = 0.02`）
- ✅ 新增：三级渐进式动态止损机制

**新特性**：

1. **监控阶段（-1.5%）**
   - 当亏损达到-1.5%时，股票进入监控观察名单
   - 记录最低盈亏比例和监控开始时间
   - 不执行任何卖出操作，仅预警

2. **一级止损（-2.5%）**
   - 触发条件：亏损达到-2.5%
   - 执行动作：卖出50%仓位
   - 目的：控制风险敞口，保留反弹机会

3. **二级止损（-4.0%）**
   - 触发条件：亏损达到-4.0%
   - 执行动作：全部清仓
   - 目的：果断止损，防止深套

4. **反弹保护机制**
   - 触发条件：已触发一级止损后，股价反弹超过成本价（盈利>0%）
   - 执行动作：重置所有止损状态
   - 价值：避免"卖在最低点"，给予优质股票二次机会

---

### 📝 配置变更

#### 修改文件：`config/settings.py`

**旧配置**：
```python
STOP_LOSS_RATIO = 0.02          # 硬止损阈值（-2%）
```

**新配置**：
```python
# 【动态分级止损】

STOP_LOSS_LEVEL1_THRESHOLD = 0.025    # 一级止损阈值（-2.5%减半仓）
STOP_LOSS_LEVEL2_THRESHOLD = 0.04     # 二级止损阈值（-4%清仓）
STOP_LOSS_CHECK_INTERVAL = 5          # 止损检查频率（每 5 秒）
STOP_LOSS_START_TIME = "1045"         # 硬止损开始执行时间（10:45 后）
STOP_LOSS_END_TIME = "1450"           # 硬止损结束执行时间（14:50 前）
ENABLE_HARD_STOP = True               # 硬止损开关
EARLIEST_EXECUTION_TIME = 952         # 动态止盈最早执行时间（09:52）
```
