# 延时策略天数设置 - 快速参考

## 📅 天数计算逻辑

### 核心公式
```
目标日期 = 信号触发日期 + delay_days (自然日)
```

### 实际例子
```
配置：delay_days = 3
信号日：2026-04-01 (周三)
目标日：2026-04-04 (周六)

执行时间轴：
Day 1 (04-01 周三): 信号触发 → 加入观察名单
Day 2 (04-02 周四): 等待中...
Day 3 (04-03 周五): 等待中...
Day 4 (04-04 周六): 目标日，但不开盘 → 继续等待
Day 5 (04-05 周日): 目标日，但不开盘 → 继续等待  
Day 6 (04-06 周一): 今天 >= 目标日 → 执行买入
```

## 🎯 设置建议

### 按股票类型
```
激进型慢涨股: delay_days = 1-2
标准慢涨股:   delay_days = 2-3  
超慢涨股:     delay_days = 3-4
```

### 按市场环境
```
牛市: delay_days - 1 (加快节奏)
震荡: delay_days (标准设置)  
熊市: delay_days + 1 (充分整理)
```

## ⚠️ 注意事项

### 1. 节假日影响
```
❌ 错误预期：
信号：04-30 (周四) + delay_days:3 = 目标：05-03 (周一)
以为05-03会执行 → 但五一假期不开盘

✅ 实际执行：
05-04 假期 → 05-05 假期 → 05-06 (首个交易日) 才执行
实际等待：6天而不是3天！
```

### 2. 信号优先原则
```
目标日期当天：
├─ 有BUY信号 → 用信号价买入 (优先，通常更低)
├─ 无信号 → 用实时价买入 (保底，确保成交)
```

## 🎯 实战技巧

### 1. 个性化设置
```
短线强势股: 1-2天
中线慢牛股: 2-3天  
长线整理股: 3-4天
```

### 2. 时间选择
```
✅ 周三前触发 → 避开周末影响
✅ 避开节假日前后
✅ 结合个股K线形态
```

### 3. 参数微调
```
如果提前触发率低 → 适当增加 delay_days
如果等待太久 → 适当减少 delay_days  
如果成本太高 → 提高 min_volume_ratio
```
