# 动态止损 - 快速参考卡（V8.95 保守版）

**版本**: V8.95 | **更新**: 2026-04-15 | **类型**: 保守型风控

---

## 🎯 核心参数（config/settings.py）

```python
STOP_LOSS_MONITOR_THRESHOLD = 0.008     # -0.8% 开始监控（更早预警）
STOP_LOSS_LEVEL1_THRESHOLD = 0.017      # -1.7% 减半仓（快速行动）
STOP_LOSS_LEVEL2_THRESHOLD = 0.035      # -3.5% 全清仓（严格控制）
STOP_LOSS_CHECK_INTERVAL = 5            # 每5秒检查
STOP_LOSS_START_TIME = "1045"           # 10:45开始
STOP_LOSS_END_TIME = "1450"             # 14:50结束
```

---

## 📊 工作流程（保守版）

```
持仓 → 亏损≥-0.8% → 【监控】→ 亏损≥-1.7% → 【一级止损卖50%】
                                          ↓
                                    反弹>成本价 → 【重置状态】
                                          ↓
                                    亏损≥-3.5% → 【二级止损清仓】
```

## 🔧 常用调整

### 切换到平衡版（V8.9）
```python
MONITOR = 0.015   # -1.5%
LEVEL1  = 0.025   # -2.5%
LEVEL2  = 0.04    # -4.0%
```

### 更极端保守（超保守）
```python
MONITOR = 0.005   # -0.5%
LEVEL1  = 0.012   # -1.2%
LEVEL2  = 0.025   # -2.5%
```

---

## 📝 日志关键词

- `[止损-监控]` - 进入监控状态（-0.8%触发）
- `[止损-一级]` - 触发一级止损（-1.7%触发）
- `[止损-二级]` - 触发二级止损（-3.5%触发）
- `[止损-重置]` - 反弹保护生效
- `[止损执行]` - 成功卖出
- `[止损失败]` - 下单失败

---

## ⚠️ 注意事项

1. ✅ 仅在10:45-14:50执行（避开开盘波动和尾盘集合竞价）
2. ✅ 反弹保护需要盈利>0%才触发（不是小幅反弹）
3. ✅ 跌停时以跌停价挂单，可能当日无法成交
4. ✅ 与动态止盈独立运行，互不冲突
5. ⚠️ **保守版特点**：监控更频繁，但实际卖出可控
