# 慢涨股延时策略 - 快速参考卡

## 📋 一句话概述
**专门捕捉信号后 2-3 天才涨停的慢涨股（如美诺华），自动过滤 SELL 信号和低量比信号。**

---

## 🎯 核心流程（3 步）

```
1. 信号到达 → 过滤 SELL 和低量比 → 写入观察名单
2. 等待 N 天 (delay_days) → 每日轮询检查
3. 到达目标日期 → 自动买入 → 从名单删除
```

## 🔍 信号处理规则

| 信号类型 | 量比 | 股票类型 | 结果 |
|---------|------|---------|------|
| BUY | ≥18 | delayed | ✅ 加入观察名单，等待 N 天 |
| BUY | <18 | delayed | ❌ 丢弃（量比不足） |
| BUY | 任意 | immediate | ✅ 即时下单（原逻辑） |
| **SELL** | 任意 | 任意 | ❌ **直接忽略（严禁写入）** |

## 🛠️ 如何配置慢涨股？

### 步骤 1：识别慢涨股特征
- ✅ 信号当天涨幅<3%，甚至下跌
- ✅ 后续 2-3 天内出现涨停
- ✅ 量比通常较高 (≥18)
- ✅ 有明显洗盘特征

### 步骤 2：编辑配置文件
打开 `data/stock_personalities.json`，添加:

```json
"股票代码": {
    "name": "股票名称",
    "type": "delayed",
    "delay_days": 2-4,          // 根据洗盘天数设定
    "min_volume_ratio": 15.0-25.0  // 高量比要求
}
```

### 步骤 3：保存生效
**无需重启程序！** 下次信号处理自动应用新配置。

## 📈 参数设置建议

### delay_days（延时天数）
| 类型 | 天数 | 适用股票 |
|------|------|---------|
| 超强势 | 0-1 | 当天或次日涨停 |
| 中等强势 | 1-2 | 洗盘 1 天后拉升 |
| **慢涨股** | **2-4** | **美诺华模式** ⭐ |
| 超慢涨 | 5-7 | 较少见 |

### min_volume_ratio（最小量比）
| 类型 | 量比 | 适用场景 |
|------|------|---------|
| 普通即时 | 1.5-2.0 | 常规信号 |
| 强势股 | 3.0-5.0 | 强势突破 |
| **慢涨股** | **15.0-25.0** | **美诺华模式** ⭐ |

## 🐛 故障排查

### 问题 1：信号没有被接收
**检查**:
1. 量比是否达到阈值？
2. 股票是否配置为 type="delayed"？
3. 是否是 SELL 信号？（会被过滤）

### 问题 2：到达日期没有买入
**检查**:
1. 现金是否充足？
2. 是否已持有该股票？
3. 是否在交易时间？
4. 查看日志中的具体错误信息

### 问题 3：配置文件修改不生效
**解决**:
1. 检查 JSON 格式是否正确
2. 等待下一次信号处理（自动重载）
3. 查看日志确认是否成功加载

## 💡 核心代码（伪代码）

```python
# 信号处理
if action == "SELL":
    return False  # 严格过滤

if volume_ratio < min_vr:
    return False  # 量比过滤

if type != "delayed":
    return False  # 不是慢涨股

# 写入观察名单
watchlist[code] = {
    'target_date': signal_date + delay_days,
    'status': 'waiting'
}

# 每日轮询
if today >= target_date:
    buy()  # 执行买入
    delete from watchlist  # 清理
```
