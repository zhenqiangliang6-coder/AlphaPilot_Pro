## 🛠️ 常见修改

### 修改止损阈值
编辑 `config/settings.py` 第50行：
```python
STOP_LOSS_RATIO = 0.08  # 改为 0.10 就是 10%
```
⚠️ 需要重启程序

### 修改量比门槛
编辑 `strategies/signal_strategy.py` 第124-134行：
```python
if vr >= 1.8: return True  # 改为 2.0 更保守
```
✅ 自动热更新，无需重启

### 修改个股延时天数
编辑 `data/stock_personalities.json`：
```json
{
    "603538": {
        "delay_days": 1  # 改为 2 就是延时2天
    }
}
```
✅ 自动热更新，无需重启
